隐私政策



《[智评商家]小程序隐私政策》



最后更新日期：[2025年9月11日]



我们深知个人信息对您的重要性，并致力于保护您的隐私。本隐私政策旨在向您说明，在您使用【智评商家】小程序时，我们如何收集、使用、存储、共享和保护您的个人信息，以及您如何行使您的个人信息权利。



一、 我们如何收集和使用您的信息



1.1 微信授权信息：当您使用微信登录时，我们会获取您的微信头像、昵称、地区，用于创建您的账户标识和提供基础服务。您可以通过微信的隐私设置管理这些信息的授权。



1.2 位置信息（如需要）：为了提供【例如：附近门店定位、地址选择等服务】，在获得您的授权后，我们会收集您的精确或粗略位置信息。您可以在设备系统或微信中关闭位置权限。



1.3 订单与交易信息：当您使用小程序进行交易时，我们会收集您的订单信息、收货地址、联系方式和支付信息（支付信息由微信支付处理，我们不直接收集您的银行卡号等敏感信息）。



1.4 设备信息：我们会收集您设备的基本信息，如设备型号、操作系统版本、唯一设备标识符（如Android ID/IDFA）、网络状态，用于保障服务的安全运行、排查崩溃问题以及优化服务体验。



1.5 日志信息：当您使用我们的服务时，我们会自动收集您对我们服务的详细使用情况，作为有关网络日志保存，例如访问的页面、点击的记录、IP地址。



二、 我们如何使用Cookie和同类技术



我们可能会使用小程序缓存技术（如localStorage）来存储您的偏好设置、购物车内容等，以提升您的使用体验。您可以在小程序设置中清除这些数据。



三、 我们如何共享、转让、公开披露您的信息



3.1 共享：我们不会与任何第三方共享您的个人信息，但以下情况除外：



• 获得您的明确同意；



• 为实现特定功能与授权合作伙伴共享：例如，为完成配送必须与物流服务商共享您的收货人姓名、地址和电话；



• 遵循法律法规：根据法律法规规定、诉讼争议解决需要，或应行政、司法机关的要求。



3.2 转让与公开披露：我们不会将您的个人信息转让给任何公司、组织和个人，也不会公开披露，法律法规要求除外。



四、 我们如何存储和保护您的信息



4.1 存储地点：我们在中华人民共和国境内运营，您的个人信息将存储在境内。



4.2 存储期限：我们只会在为实现服务目的所必需的期限内保留您的信息，除非法律要求或允许更长的保留期。



4.3 安全措施：我们采取了符合行业标准的安全防护措施（如加密传输、访问控制）来防止您的信息遭到未经授权的访问、使用或泄露。



五、 您的权利



您对自己的个人信息享有以下权利：



5.1 访问和更正：您可以在小程序内【例如：“我的-个人资料”】访问和修改您的个人信息。



5.2 删除：您可以通过【请填写您的客服联系方式】请求删除您的个人信息。在符合法律法规规定的情况下，我们会响应您的请求。



5.3 撤回同意：您可以通过小程序的【设置】或设备系统权限管理，关闭位置等权限，以撤回对我们继续收集您个人信息的授权。



5.4 注销账号：您可以通过【请说明注销账号的具体路径或联系方式】申请注销您的账户。我们将在验证您的身份后为您处理。



六、 我们如何处理儿童的个人信息



我们的服务主要面向成人。如果没有父母或监护人的同意，儿童不应创建自己的用户账户。如果我们发现自己在未获得父母同意的情况下收集了儿童的个人信息，我们会设法尽快删除相关数据。



七、 隐私政策的更新



我们可能会适时更新本隐私政策。更新后的政策将通过小程序公告等方式通知您。若您继续使用我们的服务，即表示您同意接受更新后的政策。
